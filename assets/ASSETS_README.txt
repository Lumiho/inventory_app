REQUIRED ASSETS FOR INVENTORY COUNTER APP
=========================================

Place the following image files in this folder:

1. icon.png
   - Size: 1024x1024 pixels
   - Used as the app icon
   - Square format, no transparency recommended

2. splash.png
   - Size: 1284x2778 pixels (or similar aspect ratio)
   - Displayed while app loads
   - Background color in app.json is #2563eb (blue)

3. adaptive-icon.png
   - Size: 1024x1024 pixels
   - Android adaptive icon foreground
   - Should have padding for safe area

4. favicon.png
   - Size: 48x48 pixels
   - Used for web version

QUICK START:
You can use any placeholder images to start. The app will work
without custom icons, but may show warnings.

For a simple solution, create solid color squares or use the
Expo default icons until you have custom designs.
